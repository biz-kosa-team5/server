## 메모

- `잘 나옴`은 API가 정상 응답했고 추천/비교 의도에 맞는 답변을 반환한 경우입니다.
- `애매함`은 답변은 나왔지만 후보 수가 부족하거나, 응답에 정보 부족 표현이 섞였거나, 구조화 응답 흔적이 약한 경우입니다.
- `안 나옴`은 API 실패, 잘못된 분기, 본문 없음, 또는 추천/비교로 보기 어려운 응답을 의미합니다.
- 이 문서는 실제 로컬 서버에 80개 질문을 순차 POST 호출해서 만든 결과입니다.
